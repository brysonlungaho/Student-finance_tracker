About

Student Finance Tracker - A responsive web application for tracking student expenses.

Features

  • Track transactions with categories
  • Regex-powered search with highlighting
  • Budget tracking with visual indicators
  • Multiple currency support
  • Data import/export (JSON/CSV)
  • Fully keyboard accessible

Regex Examples

  • coffee|tea - Find coffee OR tea transactions
  • \b\d{2}\.\d{2}\b - Find amounts with cents
  • \b(\w+)\s+\1\b - Find duplicate words
  • ^@\w+ - Find tags starting with @

Keyboard Shortcuts

  • Tab - Navigate through interactive elements
  • Enter - Activate buttons/links
  • Space - Toggle checkboxes
  • Esc - Cancel editing
